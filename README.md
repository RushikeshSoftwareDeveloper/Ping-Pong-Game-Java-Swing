# 🏓 Ping Pong Game (Java Swing)

## 📌 Description
A 2-player Ping Pong game developed using Java Swing. Players control paddles and try to bounce the ball to score points.

## 🎮 Controls
- Player 1 → W / S
- Player 2 → Up / Down Arrow

## 🚀 Features
- Real-time gameplay
- Ball collision logic
- Score system

## 🛠 Concepts Used
✔ Java Swing  
✔ Timer  
✔ Event Handling  
✔ Collision Detection
